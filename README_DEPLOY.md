# ClickProps - Public Buyer Portal

Built with Next.js 14, Prisma, PostgreSQL, and Konva.js for interactive floor plans.

## Features
- Public landing page with featured projects
- Properties listing with search and filters
- Interactive floor plan viewer (grid + site plan modes)
- Unit enquiry forms
- Responsive design (mobile-first)

## Deployment
Deployed on Railway with auto-scaling PostgreSQL and Redis.
